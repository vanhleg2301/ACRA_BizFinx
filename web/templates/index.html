How to use — filing workflow ▾

What this tool does: it reads a company's Word/Excel financial statements and produces an ACRA BizFinX XBRL package (a ZIP of .xbrl / .xsd / linkbase files) that you upload to the BizFinx Preparation Tool 4.0 for filing.

  1. Pick a company in the left sidebar (or click Run Demo to try the built-in EVX Ventures sample — no upload needed).
  2. Upload inputs — drop the Word financial statements (.docx) and/or the Excel figures into the Input Files boxes. The Word file supplies the full-statements text block; Excel supplies the tagged figures.
  3. Choose a mode — Pre-fill (Excel + JSON to review) or Full XBRL Package (the ZIP for filing). Leave Skip Validation off so calc/sign checks run first.
  4. Run — watch the log. Green = pass, yellow = warnings to review, red = errors to fix in the source file before re-running.
  5. Download ZIP and open it in BizFinx → Validate (offline), then Validate Online → Acknowledge and Upload.

Note: the review-copy PDF and the online validation are produced by BizFinx, not by this tool. This tool only builds the XBRL package that feeds them.
